Customize The Search Settings Of Wembra.

Safe Search

This setting will enable you to filter your search results and remove any kind of inappropriate web result or image
And this is also a very good feauture for parents that want their child to surf safely.


Results Per Page

This feature will allow you to select how many results you want on a single page as this will also effect your search speed
As if you select more reults it will take more time for us to process.



Opening Results

This feature will enable you to open results in the new window.You can use this if want and if it is unchecked
the results will be shown in the same window.
